## API de gerenciamento de pessoas


### [GET] /api/v1/person

Retorna uma lista com todas as pessoas cadastradas.

### [POST] /api/v1/person

Cadastra uma nova pessoa.

### [GET] /api/v1/person/{id}

Retorna a pessoa com o id informado.

### [PUT] /api/v1/person/{id}

Atualiza a pessoa com o id informado.

### [DELETE] /api/v1/person/{id}

Deleta a pessoa com o id informado.

Link de teste: http://localhost:8080/swagger-ui.html
